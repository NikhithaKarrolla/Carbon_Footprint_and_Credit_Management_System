# 🌱 Carbon Emission & Credit Management System

## 📌 Overview
This project helps industries track, predict, and manage carbon emissions using machine learning.

## 🚀 Features
- Carbon emission calculation
- AI-based prediction (Linear Regression)
- Carbon credit estimation
- Dashboard visualization
- Role-based system

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB
- AI: Python + Flask (scikit-learn)

## ⚙️ Setup Instructions

### Backend
cd backend  
npm install  
node server.js  

### AI Service
cd ai-service  
python app.py  

### Frontend
cd frontend  
npm install  
npm start  

## 📊 Output
- Emission prediction
- Carbon credit status
- Graphical reports
